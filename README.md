# 📊 Financial Analytics Dashboard (Power BI)

## 📌 Project Overview
This project presents a Financial Analytics Dashboard developed using Power BI for a retail store environment.

The dashboard provides a structured and interactive view of key financial statements, enabling stakeholders to monitor performance, assess financial position, and analyze receivables effectively.

---

## 🎯 Objectives
- Analyze financial performance using Income Statement  
- Evaluate financial position through Balance Sheet  
- Identify overdue receivables and financial risk using Aging Analysis  
- Support data-driven decision-making  

---

## 📊 Dashboard Modules

### 🔹 Executive Financial Overview
- High-level summary of financial insights  
- Clean and user-friendly layout  
- Navigation hub for all reports  

---

### 🔹 Income Statement Analysis
- Revenue and profitability insights  
- Gross Profit and Operating Margin KPIs  
- Revenue trend analysis  
- Expense breakdown  
- Detailed Profit & Loss structure  

---

### 🔹 Balance Sheet Overview
- Assets, Liabilities, and Equity analysis  
- Financial structure representation  
- Asset and liability breakdown  
- Organized financial reporting  

---

### 🔹 Accounts Receivable Aging
- Receivables monitoring and analysis  
- Aging bucket classification (0–30, 31–60, etc.)  
- Average and maximum aging indicators  
- Identification of high-risk overdue amounts
